/** SignCard usage

```jsx
<SignCard
  ground="indigo"
  title="NOTICE"
  ribbon="Children 12 & under"
  note="Must be accompanied by an adult when using the pool"
/>
<SignCard ground="bone" title="GOOD VIBES" ribbon="Poolside Kitchen" greca={false} />
```

The signature panel. `ground`: `indigo` / `brick` / `bone` (bone flips the
headline to terracotta-on-light). Composes `Ribbon`. The greca border draws
along the bottom edge unless `greca={false}`.
*/
